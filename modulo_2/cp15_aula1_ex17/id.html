Treinando IDs

O que é uma id?

Uma id como é um pouco explicito pelo próprio nome, é uma identificação. identificação essa que será dada a uma tag

Por exemplo, esse parágrafo, diferente do anterior está identificado por um id de nome exemplo

Já esse não. Podemos mudar o estilo de uma tag específica que se repete várias vezes. Mas diferente do class, o id só pode ser pertencente a uma tag. Nesse caso, não posso identificar esse parágrafo também com o id exemplo, mas posso identificá-lo como conconclusao por exemplo

E basicamente é isso. Só uma última coisa, os ids sobrepõem tanto as classes como a tag em si.
